# Hunty Technical Test

A technical test for Hunty.

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run generate

# pnpm
pnpm run generate

# yarn
yarn generate
```

## Deployment

Deploy the application to Firebase

```bash
# npm
npm run deploy

# pnpm
pnpm run deploy

# yarn
yarn deploy
```
